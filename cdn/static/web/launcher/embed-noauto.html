To automatically launch Zwift and skip this step create a file called "auto_launch.txt" inside zoffline's storage directory.
Warning: If there's a pending Zwift update that zoffline supports, auto launching will prevent that update from being installed and Zwift will not behave nicely. Always ensure zoffline is disabled to apply updates in advance if auto launching.
